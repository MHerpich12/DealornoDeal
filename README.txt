Simulation of Deal or no Deal gameshow in Java.  GameFrame class holds main method and initializes GUI.  DealOrNoDeal class initializes the briefcases and controls game play.  Users can choose from 24 cases and cases per round are pre-determined, declining from 6 to 1.  User wins if an offer is accepted for more than the case is worth or if banker's final offer is rejected and is less than case value.
